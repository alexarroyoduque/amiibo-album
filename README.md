# amiibo-album

Experimental project to check your amiibos. Developed with lit-html.

[https://amiibo-album.firebaseapp.com](https://amiibo-album.firebaseapp.com)

## Quick start

```
npm install -g polymer-cli
git clone this-repo
cd amiibo-album
npm install
polymer serve
```

## Deploy
```
polymer build
polymer serve build/default

```
```
firebase deploy
```
